## 🌟 About This Repository

Welcome to my **Virtual Tech CODTECH Internship** portfolio! This repository showcases all the projects and tasks I completed during my internship, spanning across three core domains: **Data Science**, **Machine Learning**, and **Artificial Intelligence**.

---

## 📂 Project Structure

<details>
<summary><b><font size="+1">📊 Data Science Tasks</font></b></summary>
<br>
This module focuses on foundational data engineering, analysis, and basic neural networks.

1. **ETL Data Pipeline Development**: Synthetic data generation, imputation, and encoding pipelines using Scikit-learn and Pandas.
2. **Deep Learning Project**: A Convolutional Neural Network (CNN) built with TensorFlow/Keras to classify handwritten digits (MNIST).
3. **End-To-End Data Science Project**: A Random Forest classification model served through a fully functional **Flask REST API** with an interactive HTML/JS web interface.
4. **Optimization Model**: A business production planning problem solved using Linear Programming and the `pulp` library.
</details>

<details>
<summary><b><font size="+1">🤖 Machine Learning Tasks</font></b></summary>
<br>
This module dives into core machine learning algorithms and Natural Language Processing.

1. **Decision Tree Implementation**: Predicting outcomes on the Breast Cancer dataset with comprehensive tree visualizations.
2. **Sentiment Analysis with NLP**: Customer review classification using TF-IDF vectorization and Logistic Regression.
3. **Image Classification Model**: Advanced CNN architecture to classify complex images.
4. **Recommendation System**: A Collaborative Filtering engine predicting user preferences via a user-item rating matrix and Cosine Similarity.
</details>

<details>
<summary><b><font size="+1">🧠 Artificial Intelligence Tasks</font></b></summary>
<br>
This module explores modern AI applications including real-time detection and interactive dashboards.

1. **Image Classification (Fashion MNIST)**: A robust CNN predicting fashion items with high accuracy and visual reporting.
2. **Real-time Object Detection**: A computer vision script leveraging **YOLOv8** to identify and bound objects in real-time via webcam.
3. **Sentiment Analysis Dashboard**: A beautiful, interactive web application built with **Streamlit** and **TextBlob** for instant NLP analysis.
4. **Content-Based Recommendation Engine**: A movie recommendation system using NLP (TF-IDF) to analyze descriptions and suggest similar films.
</details>

## 🚀 How to Run

Clone the repository and install the dependencies to test out any of the modules!

```bash
# Clone the repository
git clone https://github.com/GOPID1603/Virtual-Tech.git
cd Virtual-Tech

# Install requirements
pip install pandas scikit-learn tensorflow matplotlib pulp flask streamlit ultralytics textblob opencv-python

# Navigate to a specific task folder
cd "AI Tasks"
python task3_sentiment_dashboard.py # Example: Run the Streamlit Dashboard
```
